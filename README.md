# Cidadania Urbana

## Projeto Final — Programação para Dispositivos Móveis em Android

Aplicativo mobile desenvolvido em React Native com Expo para registro de problemas urbanos.

## Tema Social

O aplicativo ajuda cidadãos a registrarem problemas urbanos como:

- Buracos
- Falta de iluminação
- Dengue
- Lixo irregular
- Vazamentos

## Tecnologias Utilizadas

- React Native
- Expo
- React Navigation
- SQLite
- Hooks
- JavaScript

## Funcionalidades

- Criar ocorrências
- Listar ocorrências
- Editar ocorrências
- Excluir ocorrências
- Visualizar detalhes

## CRUD Completo

O aplicativo implementa:

- CREATE
- READ
- UPDATE
- DELETE

## Como Executar

### Instalar dependências

```bash
npm install
```

### Executar

```bash
npx expo start
```

## Hooks Utilizados

- useState
- useEffect

## Banco de Dados

Persistência local utilizando SQLite.
